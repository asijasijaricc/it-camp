Krecemo sa stilovima

CSS (Cascading Style Sheets) je jezik koji koristimo za stilizovanje HTML dokumenta.

CSS mozemo primeniti na 3 nacina:

  1. Inline CSS;
  2. Stavljanjem style taga unutar head;
  3. Eksterni CSS.
